About the Project

Skycoders Airline Management System is a fully integrated application that utilizes modern web technologies including HTML, CSS, and JavaScript for the user interface. It connects with a robust backend logic written in C language, compiled into WebAssembly using the powerful Emscripten compiler. This fusion of technologies allows for seamless performance and native-level speed within the browser.

The system enables users to register, log in, book flights, check status, and manage their bookings. The C backend handles core logic like data structures, file management, and memory-efficient operations, while the frontend provides a modern and responsive experience for users.
